# 贡献指南

感谢参与 Worker Aid Agent。

## 可贡献内容

- 场景数据：新增常见劳动争议类型。
- 证据清单：补充某类争议的证据材料。
- 模板：优化投诉、仲裁、法律援助申请材料草稿。
- 文档：补充本地化办事指南，但必须标注来源和更新时间。
- 代码：改进 CLI、计算器、校验器、Web UI、RAG 检索。

## 合规要求

1. 不上传真实个人信息或未脱敏案例。
2. 不提交规避法律、伪造证据、骚扰施压类内容。
3. 法律依据必须尽量引用官方来源。
4. 对不确定内容使用“需核验”“可能”“通常”等谨慎措辞。
5. 涉及具体个案策略的内容必须提示人工复核。

## Pull Request 建议

- 一个 PR 只解决一个主题；
- 新增数据时同步补充来源和更新时间；
- 修改金额计算逻辑时补充示例；
- 文书模板避免使用绝对化承诺。
