# Worker Aid 插件包说明

`worker-aid.plugin.json` 是面向专业协作者的插件包入口，用于把本仓库的 `skills/`、`agents/` 和 `prompts/` 安装到支持插件或技能目录的 AI 工具中。

## 适用对象

- 公益律师、法律援助志愿者、工会工作者和劳动者互助社区维护者。
- 已经熟悉 Codex、Claude、OpenClaw、Cursor 等 AI 工具的 power user。
- 需要在已有 AI 工作流中复核、整理、批量生成材料草稿的协作者。

普通劳动者应优先使用本地 App，而不是先学习插件安装。

## 打包内容

- `skills/`：按欠薪、加班费、证据整理、法援渠道、仲裁申请书等场景组织的技能。
- `agents/`：案情采集、证据整理、文书草拟等角色说明。
- `prompts/`：可复制到不同 AI 工具中的系统提示和流程提示。
- `docs/AGENT_PROTOCOL.md`：统一输入输出、安全边界和人工复核要求。

## 安全默认值

插件包的默认配置要求：

- 输出包含不确定性和人工复核提醒。
- 阻止伪造、补写、篡改或诱导伪造证据。
- 不冒充律师、法律援助机构、仲裁委、法院或行政机关。
- 不承诺投诉、仲裁、诉讼、调解或法律援助结果。

## 维护原则

独立本地 App 和插件包共享同一套核心数据、模板、风险拦截和安全协议。新增法律或地方政策内容时，优先引用官方来源；地方政策必须标明省/市和更新日期。
