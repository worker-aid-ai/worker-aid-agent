# 隐私与安全规则

## 数据最小化

只收集解决问题所必需的信息。默认不要求用户提供身份证号、完整手机号、家庭住址、银行卡号、公司内部账号、完整合同扫描件等敏感信息。

## 脱敏规则

公开提交案例时应使用以下格式：

| 原始信息 | 脱敏示例 |
|---|---|
| 张三 | 劳动者A |
| 深圳市南山区某公司 | 深圳某科技公司 |
| 13800138000 | 138****8000 |
| 4403xxxxxxxxxxxx | 4403************ |
| 详细住址 | 广东省深圳市某区 |

## 禁止内容

- 真实个人身份信息；
- 未经授权的公司商业秘密；
- 诱导或协助伪造证据；
- 网络暴力、人肉搜索、骚扰话术；
- 规避劳动监管或逃避法定义务的指导。

## 安全输出要求

AI 输出应始终包含：

1. 事实摘要；
2. 不确定点；
3. 证据缺口；
4. 可选择的合法路径；
5. 需要人工复核的事项；
6. 不构成法律意见的提示。
