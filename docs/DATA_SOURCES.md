# 数据源与更新说明

本项目新增两个启动版数据文件：

- `data/minimum-wage-cn-2025.json`：中国大陆各地区最低工资标准启动版数据。
- `data/service-portals-cn.json`：法律援助、劳动监察/人社服务、仲裁入口索引。

## 重要边界

1. 数据用于 AI Agent 分流、提醒和材料准备，不作为官方实时口径。
2. 最低工资标准由省、自治区、直辖市确定；同一省内通常有多个档位，需结合劳动合同履行地/用工地确认。
3. 法律援助、劳动监察、仲裁的受理范围、线上入口和材料要求会按地区变化，应以当地司法行政机关、人社部门、仲裁委最新要求为准。

## 全国入口

| 类型 | 入口 | 用途 |
|---|---|---|
| 人社服务/12333 | https://www.12333.gov.cn/ | 人社政策、办事指南、地方服务窗口、劳动关系相关服务 |
| 法律服务/12348 | https://www.12348.gov.cn/ | 公共法律服务、法律援助、在线咨询入口 |
| 中国政务服务平台 | https://gjzwfw.www.gov.cn/ | 跨地区政务服务检索入口 |

## 更新流程建议

1. 每月或每季度运行一次数据复核任务。
2. 对最低工资：优先核验省级人社厅/人社局官网、政府公报、12333 平台。
3. 对法律援助：优先核验省级司法厅/司法局、法律援助中心、中国法律服务网。
4. 对劳动监察：优先核验省级/市级人社局“劳动保障监察”“欠薪投诉举报”“根治欠薪”栏目。
5. 更新时必须记录：`sourceUrl`、`effectiveDate`、`lastReviewed`、`reviewer`、`verificationStatus`。
