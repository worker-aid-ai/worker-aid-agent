# 项目章程

## 使命

Worker Aid Agent 致力于为中国普通劳动者提供开源、可审计、可复用的 AI Agent / Skill 工具，降低劳动权益问题中的信息不对称和材料整理成本。

## 边界

项目可以：

- 帮助用户梳理事实、时间线、证据、诉求；
- 解释常见劳动权益概念；
- 提供法律援助、劳动监察、仲裁等公开流程的导航；
- 生成投诉、咨询、仲裁申请等材料草稿；
- 生成待咨询律师或法律援助机构的问题清单。

项目不可以：

- 冒充律师、法官、仲裁员、政府工作人员；
- 承诺结果或替用户作出法律判断；
- 教唆伪造证据、隐匿事实、诬告陷害；
- 鼓励人肉搜索、骚扰、暴力、非法聚集或破坏系统；
- 上传、传播未脱敏的个人敏感信息或企业商业秘密。

## 设计原则

1. **普通劳动者优先**：语言尽量通俗，流程尽量明确。
2. **事实优先**：先收集和校验事实，再讨论诉求和路径。
3. **证据优先**：所有结论都要回到证据材料。
4. **本地政策优先**：社保、最低工资、工资支付周期、法律援助条件等应核验当地规则。
5. **人工复核优先**：所有输出默认是草稿，不是最终法律意见。
