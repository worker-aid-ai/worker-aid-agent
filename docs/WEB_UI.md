# Worker Aid Agent 本地 App 说明

本地 App 是面向普通劳动者的主入口。它把案情采集、模型配置、隐私确认、材料导出和专业协作者插件说明放在同一个本地网页中，默认只在本机处理输入内容。

## 启动方式

Windows PowerShell：

```powershell
.\start-worker-aid.ps1
```

或使用通用命令：

```bash
pnpm install
pnpm run start:app
```

然后打开：

```text
http://localhost:5173/
```

## 本机草稿

本地 App 可以把当前案情保存到浏览器本机存储中，方便稍后继续整理。保存内容只应包含已经脱敏或概括后的材料摘要；API key 不会写入本机草稿，恢复草稿后如需调用外部模型必须重新填写 key 并再次勾选确认。

## 页面结构

1. **案情向导**：采集劳动者称呼、工作城市、用人单位简称、岗位、入职/离职日期、工资基数、欠薪金额、诉求和关键时间线。
2. **模型配置**：可选填写 provider、base URL、模型名和 API key。API key 只用于当前浏览器会话，不写入仓库文件。
3. **材料导出**：生成法援咨询摘要、仲裁申请书草稿、证据目录和案件时间线，支持 Markdown/HTML 下载。
4. **协作者插件**：指向 `worker-aid.plugin.json`，供公益律师、法援志愿者、工会工作者和 AI power user 在已有 AI 工具中复用 skills/agents/prompts。

## 隐私与安全边界

- 默认不主动上传用户输入。
- 调用外部模型前必须由用户勾选确认。
- 不要上传身份证号、完整手机号、住址、银行卡号、未脱敏合同或完整聊天记录。
- 检测到疑似完整手机号、身份证号或银行卡相关内容时，页面会提示先脱敏。
- 所有法律、金额、时效、证据、管辖和地方政策相关输出都只是材料整理草稿，提交前必须人工复核。

## 输出定位

本地 App 不冒充律师、法律援助机构、仲裁委、法院或行政机关，不承诺投诉、仲裁、诉讼、调解或法律援助结果，也不生成、补写、篡改或诱导伪造证据。
