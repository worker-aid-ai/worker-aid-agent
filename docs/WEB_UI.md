# Web 表单界面说明

本项目新增 `web/` 静态界面，用于演示：

- 案情采集；
- 加班费估算；
- 最低工资查询；
- 仲裁申请书、法律援助摘要、证据目录草稿生成；
- Markdown/HTML 下载。

启动：

```bash
pnpm run serve:web
```

访问：

```text
http://localhost:5173/
```

## 安全设计

当前 Web MVP 是离线优先静态页，不会主动上传用户输入。后续如接入后端或 LLM API，必须先实现：

- 身份证号、手机号、地址、公司内部账号等敏感信息脱敏；
- 用户确认后再调用外部模型；
- 明确展示“非律师意见”免责声明；
- 日志默认不保存原文案情；
- 导出文书时提示用户核验真实性和证据来源。
