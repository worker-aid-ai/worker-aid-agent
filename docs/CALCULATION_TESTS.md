# 金额计算与测试说明

项目新增 `src/lib/amounts.mjs` 和 `tests/*.test.mjs`，覆盖以下金额估算场景：

1. 工资欠付金额：按月份汇总应发、实发、差额。
2. 未签书面劳动合同二倍工资差额：默认最多按 11 个月估算。
3. 经济补偿 N：按工作年限折算补偿月数。
4. 违法解除赔偿金 2N：在“已初步认定违法解除”的假设下估算。
5. 加班费：工作日 1.5 倍、休息日 2 倍、法定节假日 3 倍的粗略估算。
6. 最低工资：地区/档位查询和低于最低工资差额检测。

运行：

```bash
pnpm test
pnpm run validate
```

## 注意

金额函数只做草算，不自动判断请求是否成立。实际案件还要核验：

- 仲裁时效；
- 工资基数；
- 当地裁审口径；
- 证据完整性；
- 社平工资封顶；
- 用人单位解除理由与程序；
- 调休、绩效、提成、补贴、扣款等特殊情况。
