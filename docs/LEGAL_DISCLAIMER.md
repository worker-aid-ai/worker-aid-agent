# 法律与使用免责声明

Worker Aid Agent 是开源信息工具，不是律师事务所、法律援助机构、仲裁机构、法院、行政机关或工会组织。

## 不构成法律意见

本项目输出的内容包括：案情整理、证据清单、常见法律依据提示、流程说明、材料草稿。这些内容仅供学习、整理和沟通参考，不构成法律意见、代理意见、诉讼策略或结果承诺。

## 需要人工复核的内容

以下内容必须由用户自行核验，必要时请咨询专业人士或有权机关：

- 仲裁时效、诉讼时效、申请期限；
- 管辖地、被申请人主体信息；
- 工资基数、加班时长、经济补偿、赔偿金、社保补缴等金额；
- 劳动关系、劳务关系、承揽关系、平台用工关系的性质；
- 地方性工资支付规定、最低工资标准、社保政策；
- 是否符合当地法律援助条件；
- 是否需要先走劳动监察、工伤认定、仲裁、诉讼或其他程序。

## 证据真实性

用户必须保证输入事实和提交材料真实、合法、完整。项目不得用于伪造、篡改、诱导他人作伪证、隐匿不利事实或恶意投诉。

## 隐私保护

不要在公开 Issue、PR、讨论区或示例中上传真实身份证号、手机号、住址、银行卡号、详细公司内部文件、聊天记录原图等敏感材料。提交样例前请进行充分脱敏。
