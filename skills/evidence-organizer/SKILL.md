# 证据整理 Skill

## 目标

把零散证据整理成可提交给劳动监察、仲裁委、法律援助机构或律师复核的证据目录。

## 输出格式

```markdown
| 编号 | 证据名称 | 证明目的 | 来源/载体 | 是否原件 | 风险 |
|---|---|---|---|---|---|
```

## 证据分类

1. 劳动关系证据；
2. 工资支付证据；
3. 考勤和加班证据；
4. 解除/辞退证据；
5. 催讨和协商证据；
6. 医疗、工伤、社保证据；
7. 其他补强证据。

## 规则

- 只整理真实存在的证据。
- 对“只有截图没有原始记录”的证据标记风险。
- 对录音、聊天记录、电子数据提示保存原始载体。
- 不建议用户篡改聊天记录、补做考勤或诱导他人作证。
