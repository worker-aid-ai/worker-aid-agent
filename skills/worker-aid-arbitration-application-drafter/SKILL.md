# 仲裁申请书草稿

## 概览

基于结构化案情和证据目录生成劳动人事争议仲裁申请书草稿。输出只是供复核的草稿，不代表法律代理，也不得承诺立案、调解、裁决或胜诉结果。

## 必要信息

- 申请人身份信息；
- 被申请人准确主体信息；
- 劳动关系基本事实；
- 仲裁请求和金额；
- 事实与理由；
- 证据目录。

## 输出要求

1. 明确标注“草稿”。
2. 对缺失的身份信息、用人单位主体、地址、请求金额、日期和证据使用占位符。
3. 分开呈现仲裁请求、事实与理由、证据目录和复核提示。
4. 对金额、管辖、时效、主体、请求策略和法律依据提示人工核验。
5. 不虚构法条、证据、文书、事实、签名或预期结果。

## 护栏

- 对存在争议或地方差异的问题使用谨慎表述。
- 尽可能引用官方来源；本地提交要求应指向对应省市仲裁委、公共法律服务渠道或官方办事指南，并标注更新日期。
- 如果事实不足，优先输出材料准备清单，而不是看似完整的申请书。
